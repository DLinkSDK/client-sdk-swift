Protocol

AppsFlyerAdapter

AppsFlyerLib support implement the adapter only when your project use dlink to take over AppsFlyerLib attribution Notice: Do not init and start AppsFlyerLib outside startAppsFlyerLib when you use AppAttribution to take over AppsFlyerLib attribution.

Declaration

protocol AppsFlyerAdapter : AnyObject
